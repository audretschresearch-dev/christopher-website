# Jung’s Archetypes

Jung’s Archetypes might be useful. How can we combine the Light archetypes into a persona? In a way that’s consistent? In a way that avoids Shadows/Waluigis?  

Not all Light archetypes can be combined — e.g. innocent+sage or jester+hero don’t make sense. Perhaps these archetypes will form a few different compatible clusters. Then, we have multiple versions of the AI — one for each persona-cluster.

Explorer+Sage: Darwin.

Creator+Ruler: Rockefeller, many examples of leaders with visions. This combo might be needed for the “master AI” in a system of AIs, or as a master-persona for deciding personas, the top-level agent that makes subcalls.
